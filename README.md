# AresBot

A bot to get some stats about Discord servers using a bot

## Wanted architecture

- By server
  - Top 5 Most talked channel
  - Users
    - Number of messages
    - Number of reaction
    - Average number of reaction
    - Top 5 reactions emojis
  - Channels
    - Number of messages
    - Top 5 users
